# Real-Time Face Recognition & Emotion Analysis System

## Overview
This project is an end-to-end machine learning system that performs:
- Face detection
- Face recognition (known vs unknown users)
- Emotion classification
- Real-time inference using webcam

## Tech Stack
- Python
- OpenCV
- TensorFlow / PyTorch (later)
- FastAPI (later)
